Pet-проект в области спортивной аналитики и прогнозирования исходов матчей NHL сезона 2025/26.

Было сделано:
- Самостоятельно сформировал и подготовил датасет по трем сезонам NHL, описывающий стилевые особенности команд с помощью 60 интерпретируемых признаков, включая метрики текущей формы, игровых стилей, календарной и логистической нагрузки, а также очных встреч и других статистически значимых факторов;
- Из 60-ти столбцов около половины уже считаются автоматически через Python;
- Проанализированы около десяти стратегий определения результатов событий, четыре из них на дистанции;
- Отдельное внимание уделил задаче прогнозирования ничьих как редкого события в условиях несбалансированных классов (draw rate около 0.26).
- Добавлены уникальные данные о результатах, которые не встречаются в открытой аналитике;
- Привязаны и сопоставлены два дополнительных датасета (с количеством очков и зарплат игроков), которые интегрированы в основной датасет.

Что планируется сделать:
